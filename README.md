# MCP - TJRJ

## Objetivo

Esse é um projeto sem fins lucrativos com o intuíto de auxíliar advogados/pessoas interessadas em direito a pesquisar de forma rápida e eficiente casos recentes relacionados a uma palavra chave na base de dados do TJRJ.

## Referência

**Link:** [https://www3.tjrj.jus.br/ejuris/ConsultarJurisprudencia.aspx]()

*OBS: O resultado da busca do TJRJ não possui uma ordem definida. Ele é organizado por dias, mas resultados do mesmo dia podem aparecer em ordens diferentes.*
